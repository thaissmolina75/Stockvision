# StockVision Pro - Prototipo Tecnológico (TFG)

StockVision Pro es el prototipo del Trabajo Final de Graduación para la Universidad Siglo 21. Es una aplicación avanzada para la gestión y control inteligente de inventarios que implementa los procesos de negocio core del sistema. Desarrollado con React, TypeScript, Vite y optimizado para Bun con persistencia de datos en Supabase.

## Estructura del Repositorio

* `/src`: Código fuente de la aplicación (componentes, rutas y lógica).
* `/supabase`: Configuración y migraciones de la base de datos.

## Instrucciones de Ejecución Local

Para ejecutar este proyecto de forma local, asegúrate de tener instalado Bun y sigue estos pasos:

1. Clonar el repositorio:
   ```bash
   git clone [https://github.com/thaissmolina75/Stockvision.git](https://github.com/thaissmolina75/Stockvision.git)
   cd Stockvision

```

2. Instalar dependencias:
```bash
bun install

```


3. Iniciar el entorno de desarrollo:
```bash
bun run dev

```
